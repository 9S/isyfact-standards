== Ziel der Anleitung zu Datenflussdiagrammen

Eine Anwendungslandschaft besteht aus mehreren Anwendungssystemen, die Daten untereinander und mit ihren Nutzern austauschen. Die Anleitung zu Datenflussdiagrammen hilft bei der Erstellung von Datenflussdiagrammen und gibt Best Practices zur strukturierten Beschreibung dieses Datenflusses.
